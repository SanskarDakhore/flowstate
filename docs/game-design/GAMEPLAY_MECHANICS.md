# Gameplay Mechanics

## Input Types
| Input | Action | Examples |
|-------|--------|---------|
| Tap | Quick interaction | Collect energy, activate objects |
| Swipe | Directional control | Change path, dodge obstacles |
| Hold | Sustained effect | Charge interactions, sustained activation |

## Core Systems
- **Energy** — Collectible resource feeding into scoring
- **Combo** — Consecutive interaction chain multiplier
- **Flow State** — Sustained combo reward state
- **Harmony** — Cumulative performance metric driving world transformation
