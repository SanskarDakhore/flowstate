# Core Gameplay Loop

```
Move → Interact → Collect Energy → Build Combo → Increase Harmony → Transform World → Repeat
```

## Loop Detail
1. **Move** — Player entity moves continuously along flowing paths
2. **Interact** — Tap, swipe, or hold to engage with world objects
3. **Collect** — Gather energy from interactions and environment
4. **Chain** — Build combos through consecutive successful interactions
5. **Flow** — Sustained combos activate flow state (enhanced feedback)
6. **Harmonize** — Cumulative performance increases world harmony
7. **Transform** — World visually transforms from fragmented to alive
8. **Score** — Session ends with score breakdown
