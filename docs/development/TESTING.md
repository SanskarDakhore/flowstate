# Testing Strategy

## Frontend (Unity)
- EditMode tests for pure C# logic
- PlayMode tests for MonoBehaviour integration
- Located in `frontend/Assets/Tests/`

## Backend
- Unit tests in `backend/tests/unit/`
- Integration tests in `backend/tests/integration/`
- E2E tests in `backend/tests/e2e/`
- Run with `npm test`

## Cross-System
- Contract tests in `tests/contract/`
- Integration tests in `tests/integration/`
