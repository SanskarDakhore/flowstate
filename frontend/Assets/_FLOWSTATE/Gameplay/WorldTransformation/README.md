# World Transformation

Environmental transformation system — the visual heart of FLOWSTATE.

## Responsibility

- Transform the world from fragmented/muted to alive/beautiful based on harmony
- Manage transformation zones and their visual states
- Coordinate with Presentation layer for actual visual changes
- Track which areas have been restored

## Rules

- World transformation is driven by Harmony system data
- This system defines WHAT transforms, not HOW it looks (Presentation handles visuals)
- Transformation should feel gradual and satisfying, not instant
- "The better you play, the more alive and beautiful the world becomes"
