# Harmony

Harmony and world restoration system — core to FLOWSTATE's identity.

## Responsibility

- Track the harmony level of the world during a session
- Determine which world regions transition from fragmented to restored
- Provide harmony data to World Transformation and Presentation systems

## Rules

- Harmony is the primary measure of "how well the player is playing"
- Higher harmony = more alive and beautiful the world becomes
- Harmony is derived from gameplay actions, not directly set by presentation
- This is the core feedback mechanism of FLOWSTATE
