# Adaptive Music

Adaptive music system that responds to player performance and harmony level.

## Responsibility

- Layer music tracks based on harmony/flow state
- Crossfade between intensity levels
- Synchronize music transitions with gameplay moments

## Rules

- Music should feel like a natural extension of the world transformation
- Low harmony = sparse, muted layers; High harmony = full, vibrant layers
- Transitions must be musically coherent (beat-synced when possible)
