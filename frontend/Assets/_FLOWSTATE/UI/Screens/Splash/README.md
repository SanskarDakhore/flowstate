# UI Screens

Each subdirectory contains the UI implementation for one screen.

## Screens

| Screen | Purpose |
|--------|---------|
| Splash | Loading/branding screen |
| MainMenu | Primary navigation hub |
| Gameplay | In-game HUD and overlays |
| Results | Post-session score breakdown |
| Profile | Player profile and stats |
| Leaderboard | Rankings display |
| Cosmetics | Cosmetic shop and selection |
| Settings | Audio, haptics, account settings |
| ModeSelection | Game mode picker |
