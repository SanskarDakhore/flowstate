# Domain

Domain entities and business rules, organized by bounded context.

Each subdirectory represents a domain area:
- `player/` — Player identity and profile
- `progression/` — XP, levels, unlocks
- `leaderboard/` — Rankings and competitive scoring
- `cosmetics/` — Visual customizations
- `inventory/` — Player-owned items
- `game-session/` — Session lifecycle
- `game-result/` — Score submission and validation
