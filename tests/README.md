# Cross-System Tests

Root-level tests for cross-system behavior.

## Structure
- `contract/` — API contract tests (client-server agreement)
- `integration/` — Cross-system integration tests
- `e2e/` — End-to-end tests
- `performance/` — Performance benchmarks

## Note
- Frontend-specific tests are in `frontend/Assets/Tests/`
- Backend-specific tests are in `backend/tests/`
- Root tests focus on cross-system behavior only
