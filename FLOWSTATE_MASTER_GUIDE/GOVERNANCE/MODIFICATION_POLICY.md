# Modification Policy & File Classification

Every file in the FLOWSTATE repository falls into one of 6 modification categories:

| Category | Description | Modification Rules | Examples |
|---|---|---|---|
| **Immutable** | Core Canon & Laws | Never edit during code tasks; readable reference only | `98_CANON/*` |
| **Critical** | Core Physics & Spline Math | Requires ADR, user approval, and regression testing | `movement-engine.ts`, `spline-calculator.ts` |
| **Protected** | Design Tokens & System Schemas | Edits require backwards compatibility verification | `tokens.css`, `movement-intent.ts` |
| **Conditional** | UI Screens, Ecosystem, Audio | Edits allowed if 10-Point Quality Gate passes | `player-hud.ts`, `environment-view.ts` |
| **Safe** | GDOS Specs, Tests, Documentation | Free modification for task completion | `FLOWSTATE_MASTER_GUIDE/*`, `*.test.ts` |
| **Generated** | Build artifacts & node modules | Auto-generated by tools; do not manual edit | `dist/*`, `package-lock.json` |
