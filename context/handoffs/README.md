# Handoffs

Agent-to-agent handoff documents for work that crosses domain boundaries.

## When to Create a Handoff

Create a handoff when:
- Work in your domain requires changes in another agent's domain
- You've completed a phase that enables work in another area
- A decision in your domain affects another domain's approach

## Process

1. Copy `HANDOFF_TEMPLATE.md`
2. Fill in all sections
3. Name the file: `YYYY-MM-DD_from-agent_to-agent_topic.md`
